About.

Perfect Font is a simple tool which helps you to test hundreds of fonts in realtime within your live previewed web-project. It was build to take out the pain of choosing a font that really fits your needs
(even used to create this page lately).



It provides realtime testing how you'd suspect within your prefered word processor.
Choose out of 650+ fonts provided by Google Fonts API to customize your page as fast as never before.



Check out the Github Repository if you want to learn more about How-To-Use, Features, Configuration, Contributing and Custom Builds.
